ğŸ§± 2. Partitioning Disks

Using fdisk (for MBR-style partitions):

sudo fdisk /dev/sdX
# n -> new partition
# p -> primary
# w -> write changes

Using parted (for GPT and scripting):

sudo parted /dev/sdX
(parted) mklabel gpt
(parted) mkpart primary ext4 1MiB 100%
(parted) quit
